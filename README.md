# 💬 Private Messenger

Приватный мессенджер для Android — связь между двумя устройствами через зашифрованный MQTT-протокол. Работает в России без VPN.

## Возможности

- 📨 Отправка текстовых сообщений
- 📷 Отправка фото из галереи
- ✓✓ Статус доставки сообщений (галочки)
- 🔔 Звуковые уведомления и фоновая работа
- 🌙 Светлая и тёмная тема
- 🔐 Подключение по секретному коду — только два устройства
- 📡 MQTT over TLS (порт 8883) — трафик выглядит как обычный HTTPS

## Как работает

Оба устройства подключаются к публичному MQTT-брокеру [HiveMQ](https://www.hivemq.com) через зашифрованное соединение. Секретный код генерирует уникальный топик — посторонние не могут читать переписку.

```
Устройство А  →  публикует в топик pm_ХХХХ/a
               →  подписан на топик pm_ХХХХ/b

Устройство Б  →  публикует в топик pm_ХХХХ/b
               →  подписан на топик pm_ХХХХ/a
```

## Сборка APK

Проект собирается автоматически через GitHub Actions — Android Studio не нужен.

1. Загрузи файлы проекта в репозиторий GitHub
2. Перейди во вкладку **Actions → Build APK → Run workflow**
3. Через ~5 минут скачай готовый APK из раздела **Artifacts**
4. Установи APK на оба телефона (нужно разрешить установку из неизвестных источников)

## Первый запуск

1. Открой приложение на **обоих** телефонах
2. Введи своё имя (разное на каждом устройстве)
3. Введи **одинаковый** секретный код на обоих устройствах (минимум 4 символа)
4. Нажми «Начать» — готово

## Требования

- Android 8.0 (API 26) и выше
- Доступ в интернет

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Язык | Kotlin |
| Протокол | MQTT 3.1.1 over TLS |
| Брокер | HiveMQ Public Broker |
| MQTT-клиент | Eclipse Paho |
| UI | Material Components |
| Фоновая работа | Android Foreground Service |

## Конфиденциальность

- Сообщения передаются через публичный брокер HiveMQ
- Соединение защищено TLS-шифрованием
- Секретный код хранится только на устройстве
- Сообщения **не сохраняются** на сервере после доставки
- Для максимальной приватности можно заменить брокер на собственный
